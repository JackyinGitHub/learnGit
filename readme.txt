Git is a version control system.
1)create git repository
	mkdir learnGit
	cd learnGit
	git init
2)add file to git repository
	sudo vim readme.txt
	git add readme.txt
3)commit the file
	git commit -m "wrote a reame file"
4)show status
	git status
5)show difference
	git diff
5)show history record
	git log
